<OL START=4>
	<LI><H2 CLASS="western">Product of 3 Numbers</H2>
</OL>
<P STYLE="margin-top: 0.06in">You are given a number <FONT FACE="Consolas, serif"><B>num1</B></FONT>,
<FONT FACE="Consolas, serif"><B>num2</B></FONT> and <FONT FACE="Consolas, serif"><B>num3</B></FONT>.
Write a PHP script that finds if <FONT FACE="Consolas, serif"><B>num1
* num2 * num3</B></FONT> (the product) is <B>negative</B> or
<B>positive</B>. Try to do this <B>WITHOUT</B> multiplying the 3
numbers.</P>
<P STYLE="margin-top: 0.06in">The input comes as parameters named
<FONT FACE="Consolas, serif"><B>num1</B></FONT>, <FONT FACE="Consolas, serif"><B>num2</B></FONT>
and <FONT FACE="Consolas, serif"><B>num3</B></FONT>.</P>
