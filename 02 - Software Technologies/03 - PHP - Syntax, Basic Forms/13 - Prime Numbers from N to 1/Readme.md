<P STYLE="margin-top: 0.06in">You are given a number <FONT FACE="Consolas, serif"><B>num</B></FONT>.
Write a PHP script that prints<B> </B>only the <B>prime</B> numbers
from <FONT FACE="Consolas, serif"><B>num to 1</B></FONT>. A prime
number is a number that can be divided only by 1 and itself. <B>1 is
not a prime number.</B> The input comes as a parameter named <FONT FACE="Consolas, serif"><B>num</B></FONT>.
The parameter <FONT FACE="Consolas, serif"><B>num</B></FONT> will
hold a positive integer.</P>
