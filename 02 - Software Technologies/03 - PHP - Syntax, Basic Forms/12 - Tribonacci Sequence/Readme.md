	<LI><H2 CLASS="western">Tribonacci Sequence</H2>
</OL>
<P STYLE="margin-top: 0.06in">In the <B>“Tribonacci” sequence</B>,
every number is formed by the <B>sum of the previous 3</B>.</P>
<P STYLE="margin-top: 0.06in">You are given a number <FONT FACE="Consolas, serif"><B>num</B></FONT>.
Write a PHP script that prints<B> </B><FONT FACE="Consolas, serif"><B>num</B></FONT><B>
</B>numbers from the Tribonacci sequence, each on a new line,
starting from 1. The input comes as a parameter named <FONT FACE="Consolas, serif"><B>num</B></FONT>.<SPAN LANG="bg-BG">
</SPAN>The value <FONT FACE="Consolas, serif"><B>num</B></FONT> will
always be positive integer.</P>
