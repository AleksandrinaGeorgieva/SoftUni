<OL START=4>
	<LI><H2 CLASS="western">Three Integers Sum</H2>
</OL>
<P STYLE="margin-top: 0.06in">Write a JavaScript program, which
receives <B>three numbers</B>, as a <B>string array</B>. Your task is
to check whether there exists a number in the sequence, which is
equal to the <B>sum</B> of the other two. If they are, print the
numbers and their sum in the following format: “<FONT FACE="Consolas, serif"><B>`${num1}
+ ${num2} = ${sum}`</B></FONT>”. If there’s no such element,
print “<FONT FACE="Consolas, serif"><B>No</B></FONT>”.</P>
